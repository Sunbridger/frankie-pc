## Icon 图标

### props

| 属性 | 类型           | 必填  | 默认 | 说明                         |
| ---- | -------------- | ----- | ---- | ---------------------------- |
| size | string, number | false | 12   | 图标的尺寸                   |
| type | string         | true  | -    | 图标的类型, 详见右侧所有图标 |


:::demo 图标
使用说明

```html
<template>
    <f-icon size="30" type="play"></f-icon>
</template>
```
:::
